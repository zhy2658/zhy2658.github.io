### 找我们需要哪些准备？
- 首先，请我们团队抬棺需要提前准备好相应的BGM,最好把播放BGM的设备放在地窖三天以上，防止在举行抬棺仪式时放BGM因设备辐射过重导致尸变
- 进行抬棺仪式时请无关人员远离10米以上，这有助于提升仪式的效果
- 还有提前准备好必要的一些东西，西服10套、墨镜10副、黑帽10顶、黑手套10副、一个手杖。
